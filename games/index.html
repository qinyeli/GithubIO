This page showcases some of the mini games that I built in Unity.

GuruGuru

GuruGuru is a platformer game where players solve puzzles by rotating the game world (and thus manipulating gravity)! It's the first ever original game that I made.

Evan's Adventure

Evans' adventure is a simple item-collecting platformer game where the player get to switch between an icy cold world and a warm fruitful world while proceeding. It was originally made to greet a friend visiting California from the Mid West.
